Que podemos hacer con JavaScript?


MINI TUTORIAL ATRIBUTOS HTML

  • Abre tu editor de código y crea un nuevo archivo HTML.
  • En el cuerpo del archivo HTML (<body>), agrega un elemento HTML con un atributo que quieras modificar. Por ejemplo, puedes agregar una imagen con un atributo src y un id único.
  • Debajo del elemento HTML, agrega un botón que usaremos para disparar la modificación del atributo.
  • Entre las etiquetas del archivo HTML, agrega una etiqueta <script>.
  • Dentro de la etiqueta <script>, define una función llamada changeImage que seleccionará el elemento de imagen y cambiará su atributo src.
  • Guarda y abre el archivo HTML en un navegador web.
  • Haz clic en el botón y observa cómo cambia la imagen.
