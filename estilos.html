Cambiar estilos CSS con JavaScript

Para cambiar los estilos CSS con JavaScript, se puede utilizar la propiedad style del elemento HTML que se quiere modificar.

Se puede utilizar por ejemplo :document.getElementById("miElemento").style.backgroundColor = "red";. De esta forma, se puede modificar cualquier propiedad CSS de un elemento HTML utilizando JavaScript.
